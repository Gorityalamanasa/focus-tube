# 🚀 FocusTube AI

FocusTube AI is a Chrome Extension designed to help users stay focused while using YouTube by filtering distracting content and highlighting relevant videos based on selected learning modes.

---

## ✨ Features

* 🎯 Focus Modes (Study, AI, Tech, Custom)
* 🟢 Highlights relevant videos (Green tagging)
* 🚫 Filters distracting content
* ⚡ Real-time DOM-based filtering
* 🧠 Topic-based classification (DSA, ML, Web Dev, etc.)

---

## 🛠 Tech Stack

* JavaScript (Vanilla)
* Chrome Extension (Manifest v3)
* Vite

---

## ⚙️ How it Works

1. Content script runs on YouTube pages
2. Extracts video titles dynamically
3. Matches titles with selected topics
4. Tags videos as:

   * 🟢 Relevant
   * 🔴 Distracting
5. Enhances user focus by filtering distractions

---

## 🚀 Installation

1. Clone the repository
2. Run:

   ```bash
   npm install
   npm run build
   ```
3. Open Chrome → Extensions → Enable Developer Mode
4. Click **Load Unpacked**
5. Select the build folder (`dist_chrome` or `output_*`)
